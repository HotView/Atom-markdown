python有五种基本的数据类型
- 字符串
- 整数
- 浮点数
- 布尔值
- 复数

## 为什么区分不同的类型
- 因为不同的数据类型运算规则是不同的，也就是运算符重载。
- 不同对象在内存中表示表示不同
## 额外的数据类型
Fraction（分数类）
`from fractions import Fraction`
```p
a = Fraction('1/2')
b = Fraction('2/3')
print("total is ",a+b)
```
