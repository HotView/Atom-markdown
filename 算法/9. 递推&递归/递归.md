## 定义
指在函数的定义中使用函数自身的方法。
## 思想
```py
if 问题足够简单：
	直接解决问题
	返回解
else：
	将问题分解为与原问题同构的一个或更多个更小的问题
	逐个解决这些更小的问题
def fib(n):
	if n==1 or n== 2:
		return 1
	else:
		return fib(n-1)+fib(n-2)
```
