## 概念
SQL（structured query language）结构化查询语言
ANSI标准：不同版本的SQL语言，它们必须以相似的方式共同的方式共同的支持一些主要的关键字（SELECT，UPDATE，DELETE，INSERT，WHERE），也有自己的一些私有的扩展。
## 能做什么
- 面向数据库执行查询（WHERE）
- 从数据库取回数据（SELECT）
- 插入数据
- 更新数据库中的知识
- 删除数据
## 执行语句
SELECT * from Table
