可扩展标记语言，W3C发布的。

现实生活中大量有关系的数据，层次关系。

计算机如何处理有关系的数据？
## XML语言
不但可以获取数据，还可以获得关系。
在XML语言中可以自定义标签，标签分为开始标签和结束标签。
标签里面又可以包含标签。
```
<中国>
...
...
..
</中国>
```
## 应用
- 软件的配置文件
```
<soft>
	<a>
		<a1></a1>
		<a2></a2>
	</a>
	<b>
		<b1></b1>
		<b2></b2>
	</b>
</soft>
```
- 描述程序模块之间的关系
